# Code of Conduct

This project expects respectful, constructive behavior in all interactions.

## Our Standards

- Be kind and professional.
- Assume good intent; ask clarifying questions.
- Focus on the problem, not the person.
- Keep discussions on-topic and welcoming.

## Enforcement

If you experience or witness unacceptable behavior, please open a GitHub issue requesting a private contact channel.

(If you prefer a more formal policy, consider adopting the Contributor Covenant: https://www.contributor-covenant.org/)
